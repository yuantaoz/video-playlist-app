## Instructions

#### (1) database

  (a) Create your own MongoDB instance on https://www.mongodb.com/atlas/database

  (d) Edit .env file and add it to /server

  Contact me and I will give you my .env file

#### (2) backend

    cd server
    npm start

#### (3) frontend

    cd client
    npm run dev
